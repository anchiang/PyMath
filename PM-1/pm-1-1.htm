這一講要介紹使 CoCalc
顧名思義，就是協同雲端計算
我們要在這平台使用 Python 語法，
來進行數學與科學的探討。
不同於很多線上程式課以程式主題
本講會更著重於以數學、科學為主體，
以專案方式來用程式解決處理問題。
首先搜尋 CoCalc 點選進入，
先用臉書或 Google 註冊帳號。
註冊帳號後，開啟一個新專案 PyMath
先建立一個資料夾 PyMath
在建立一個資料夾 PM-1
裡面有支援很多檔案，
先取個檔名 pm-1-1-ipynb
點選 Jupyter Notebook
I 是 Interactive
他是個互動計算筆記本
他由很多 Cell 組成。
Cell 有分三種
Code, Markdown, Raw
Code 是可計算執行的
例如，輸入 9*4*8*7
接著要執行此 Cell 要輸入 shirt + Enter
就會得到其結果 2016
Raw 是純文字格式，可以隨意書寫
Markdown 就是格式化文字
例如，使用 ** 包住文字就可以變斜體
使用 $$ 包住就可以變成 LaTeX 數學式
利用 ## 開頭就可以進入標題模式
利用 * 開頭就進入條列模式
可以在參考網路上更多 MarkDown 與 LaTeX 介紹
Python 有很多現成的套件可使用，
使用 import matplotlib
就可招換些
威力強大的特殊技
在這例子使用 plot 來繪製圖形。
從這可看到 Python 語法的簡潔性
你可以先自己試著解讀看看，
之後會再陸續介紹這些語法功能。
